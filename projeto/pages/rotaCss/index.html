O que é CSS?

CSS é a sigla para Cascading Style Sheets (ou Folha de Estilos em Cascata, traduzido para o Português). É uma linguagem de formatação de conteúdo muito utilizada em conjunto com o HTML na construção de páginas da internet, sendo responsável por toda parte “visual” do site.

O CSS foi desenvolvido em meados da década de 90 para solucionar um problema criado pela expansão da própria internet. Com a popularização da rede, as páginas foram ficando cada vez mais complexas, e os recursos do HTML já não eram mais suficientes para atender às necessidades dos desenvolvedores.

Sua primeira função foi a de separar os códigos de estruturação e apresentação de todo o conteúdo, tornando-os mais simples de serem escritos e atualizados, quando necessário. Dessa forma, ao invés de utilizar tags e elementos de HTML ou XML para criar cores, fontes e layout, eles passaram a ser configurados nas folhas de estilo, ou seja nosso CSS3.

Para que serve o CSS?

O CSS pode ser utilizado para estilizar um documento de texto por exemplo, alterando a cor e tamanho dos títulos e links. Pode ser usado para criar layout por exemplo, transformando uma simples coluna de texto em um layout com uma área de conteúdo principal e um sidebar (barra lateral) para as informações relacionadas. Pode até ser usado para efeitos como animações, por exemplo.

Abaixo estamos listando algumas das grandes vantagens da aplicação do CSS:

  • Aplicação de técnicas mais sofisticadas de desenvolvimento.

  • Possibilidade de manter a mesma formatação em diferentes navegadores

  • Possibilidade do controle do layout de vários documentos a partir de um único arquivo CSS

  • Aplicação de layouts diferentes de acordo com o dispositivo utilizado, possibilitando o layout responsivo.

  • Menor consumo de banda e melhor desempenho devido ao reuso do mesmo código de formatação em várias páginas

Propiedades mais utilizadas CSS

Abaixo estão as propriedades mais utilizadas e conhecidas:

Atributos O que ele altera
background cor de fundo
border adiciona borda e suas respectivas configurações
text-align posição do texto (podendo ser: start, end, center e justify.)
font-size configura tamanho da fonte
width determina a largura da área do contéudo de um elemento
height determina a altura da área do contéudo de um elemento
