O que é lógica de programação?

lógica de programação é onde aplicamos todos os conceitos de algoritmos, a definição do passo a passo e transferimos toda a lógica do algoritmo desenvolvido para uma linguagem de programação.
Em linhas gerais, lógica de programação é todo conjunto de regras e conceitos que precisamos aplicar para criar
códigos que serão interpretados e executados por um computador.
É a base fundamental para criar e resolver problemas na programação de computadores. Ela envolve a estruturação
de instruções e algoritmos de maneira clara e coerente, permitindo que um computador compreenda e execute tarefas específicas.
Essa habilidade transcende as linguagens de programação, focando na organização lógica das instruções.

Qual a importância de aprender lógica de programação?

A lógica de programação é o pilar principal para se desenvolver todas as outras habilidades de programação.
Ela é a cola que une os conceitos e permite integrar os conceitos matemáticos com o raciocínio lógico, ambos
aplicados às linguagens de programação.
Aprender a lógica de programação é essencial para construir soluções únicas, criativas e eficazes para problemas
que venham a surgir. Pessoas que se propõem a aprenderprogramação mas pulam a etapa de lógica de programação
não terão base o suficiente para conseguir resolver essas situações. Assim, elas ficarão fadadas a usar
pedaços de blocos prontos e soluções pré-formatadas.
Para além da carreira em programação, a lógica de programação é uma habilidade muito útil no cotidiano,
afinal de contas, ela auxilia a organizar melhor os pensamentos
e a estruturar os eventos de maneira mais coerente, tornando qualquer âmbito mais produtivo e eficaz.

O que é preciso?

Para conseguirmos o objetivo de instruir o computador a fazer alguma coisa, precisamos de uma linguagem de
programação, que é um meio estruturado para passar instruções para a máquina.
Para fazer isso, temos à disposição uma gama muito grande de linguagens como:
C, Java, PHP, Python, JavaScript, entre outras. Cada uma delas tem suas
características, particularidades e cenários de aplicação.

À medida que se pratica, a habilidade em lógica de programação se aprimora. Com mais experiência,
os programadores conseguem abordar problemas complexos de forma organizada e eficiente,
criando soluções que atendam às necessidades dos usuários e ao funcionamento do software.

Em essência, a lógica de programação envolve a criação de sequências de passos lógicos e precisos
para resolver um problema. Isso inclui:

  1. Definição do Problema: Comece entendendo claramente o problema que você deseja resolver.
    Isso envolveentender os requisitos, limitações e objetivos do projeto.
  2. Divisão em Etapas Menores: Divide o problema complexo em partes menores e mais gerenciáveis.
    Cada parte pode ser tratada como uma subtarefa ou etapa do algoritmo.
  3. Sequenciamento: Determine a ordem em que as etapas devem ser executadas.
    Isso geralmente envolve estabelecer uma sequência lógica de ações que levarão à solução do problema.
  4. Estruturas de Decisão: Determine onde decisões precisam ser tomadas com base em certas condições.
    Use estruturas de decisão, como declarações "if (se)" e "else (senão)", para executar diferentes ações dependendo das circunstâncias.
  5. Estruturas de Loop: Se houver partes repetitivas em sua solução,
    use estruturas de loop para executar essas partes várias vezes, economizando código e tornando o algoritmo mais eficiente.
  6. Exemplos incluem os loops "for (para)", "while (enquanto)" e "do (faça) / while (enquanto)".
  7. Manipulação de Dados: Determine como os dados serão manipulados ao longo do processo.
    Isso pode envolver cálculos, armazenamento temporário de informações e atualizações de variáveis.
  8. Pseudocódigo ou Fluxograma: Antes de escrever código real, muitos programadores preferem esboçar um
    pseudocódigo (uma descrição em linguagem humana das etapas do algoritmo) ou criar um fluxograma (um diagrama visual que representa a lógica do algoritmo).
